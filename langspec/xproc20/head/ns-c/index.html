XProc Step Namespace

1 Introduction

This document describes the names that are defined in the http://www.w3.org/ns/xproc-step by the XProc: An XML Pipeline Language specification (May, 2010 version). For updated information, please refer to the latest version of the specification.

This document describes the names that are defined in this namespace at the time of publication. The W3C reserves the right to define additional names in this namespace in the future. Language implementors and pipeline authors may amend this namespace.
